# 🎬 Netflix Recommendation System

This is a content-based movie recommendation system built using Machine Learning.  
It recommends movies based on similarity using cosine similarity.

---

## 🚀 Features
- Movie recommendation based on similarity
- Fast and efficient algorithm
- Easy to use interface (Streamlit UI)

---

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

---

## 📂 Project Structure
netflix-recommendation-system/
│── app.py
│── movies.csv
│── similarity.pkl
│── requirements.txt
│── README.md

---

## ▶️ How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
